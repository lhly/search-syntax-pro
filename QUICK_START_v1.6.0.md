## 🚀 快速体验新功能

### 1️⃣ 使用搜索模板 (30秒)

**步骤**:
1. 点击扩展图标打开面板
2. 按 `Ctrl+T` 或点击"使用搜索模板"按钮
3. 选择一个模板,如"学术论文搜索"
4. 模板参数自动填充到表单
5. 输入关键词,点击搜索

**效果**: 无需学习复杂语法,一键应用专业搜索配置!

---

### 2️⃣ 体验智能推荐 (30秒)

**步骤**:
1. 在关键词输入框输入: `机器学习论文`
2. 查看上方出现的智能推荐面板
3. 点击推荐的 `filetype:pdf` 建议
4. 自动应用 PDF 文件过滤
5. 继续点击其他推荐优化搜索

**效果**: AI级别的搜索优化建议,自动提升搜索精确度!

---

### 3️⃣ 使用键盘快捷键 (1分钟)

- `Ctrl+K` - 快速聚焦到关键词输入框

## 🔧 进阶使用

### 创建自定义模板

1. 配置好搜索参数
2. 打开模板选择器 (`Ctrl+T`)
3. 点击"保存为模板"
4. 填写模板名称和描述
5. 选择分类
6. 保存

### 自定义快捷键

1. 打开设置页面
2. 找到"快捷键设置"
3. 点击要修改的快捷键
4. 按下新的组合键
5. 自动检测冲突
6. 保存

### 导入/导出模板

**导出**:
1. 打开模板选择器
2. 找到要导出的模板
3. 点击"导出"按钮
4. 复制 JSON 内容

**导入**:
1. 打开模板选择器
2. 点击"导入模板"
3. 粘贴 JSON 内容
4. 确认导入

## ❓ 常见问题

### Q: 智能推荐不准确怎么办?
A: 智能推荐基于关键词模式和历史记录,使用越多越准确。您也可以在设置中调整推荐配置。

### Q: 快捷键冲突怎么办?
A: 系统会自动检测冲突。您可以在设置中自定义快捷键,避免与其他扩展冲突。

### Q: 如何分享我的模板?
A: 使用"导出模板"功能生成 JSON,分享给他人导入即可。

### Q: 模板使用次数有什么用?
A: 常用模板会自动排在前面,提升使用效率。
